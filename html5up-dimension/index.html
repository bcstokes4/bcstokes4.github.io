Savory Scoot

Some functionalities include: Image Storage via AWS S3, Cart Storage via Local Storage, and displaying restaurant location via Google Maps API.

SportHub

SportHub, a full-stack web application is one of my more recent personal projects. Drawing some inspiration from sites such as Etsy and Academy Sports, SportHub was created so that users can both post products for sale as well as purchase products from other users. In just a few weeks, I was able to create a fully functioning site utilizing Flask for request handling, SQLAlchemy for the database mapping and Alembic for the database version control. The UI was implemented from React, creating various components and storing data in both Redux store states as well as localStorage when necessary. The application has added levels of security through the use of various middleware. This includes middleware for protecting AWS functionality, client side and server side middleware for request data, as well as CSRF protection and user authentication.

  • Backend - Python, Flask, PostgreSQL
  • Frontend - React, Redux
  • Other Tools - AWS S3, Docker, Alembic, Render
